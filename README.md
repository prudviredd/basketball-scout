# Basketball Scout V4.1

Adds:
- Player autocomplete suggestions
- Fuzzy correction for common player-name mistakes
- Today's games dashboard
- Exact game-by-game stat logs
- Voice search

Run:
cd ~/Downloads/basketball_dashboard_v41
echo 'BALLDONTLIE_API_KEY=your_key_here' > .env
pip3 install -r requirements.txt
python3 app.py

Open:
http://127.0.0.1:5050

Try:
cad
Cade Cunningham last 5 games
Stephen Curry last 5 games threes points
